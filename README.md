# wear-os-temperature-reporter
Android Wear OS app for reporting and tracking temperature with visual health status indicators.

## Project Overview
This app allows users to enter temperature readings and will calculate the average temperature. The average is displayed dynamically and updates every time a new value is entered.  
Four images visually represent the average temperature:

- No temperature data yet  
- Normal temperature  
- Fever  
- High fever  

## Key Features
- Displays changing status images corresponding to temperature ranges  
- Calculates and updates average temperature dynamically  
- Accepts decimal temperature input in Fahrenheit
